# GHz Band Device Detector, Acoustic Fluctuation Detector, WiFi Interference Targeting, and Router Antenna Jammer System

This project contains scripts to:
1. Detect wireless devices emitting GHz band signals approximately 2-4km around your location that are not connected to your current router.

## How It Works

### WiFi Device Detection
1. The script scans for all nearby WiFi networks using Windows `netsh` command
2. It identifies the network you're currently connected to
3. For each detected network, it:
   - Determines if it's a GHz band device (2.4GHz or 5GHz)
   - Estimates the distance based on signal strength
   - Filters out devices connected to your router
   - Filters devices to only those within 2-4km range
4. Displays the results

### Acoustic Fluctuation Detection
1. Monitors WiFi signal strength fluctuations over time
2. Detects variations that may indicate acoustic wave interactions
3. Analyzes patterns in signal variations to identify potential sound sources
4. Provides likelihood assessment of acoustic activity

### WiFi Interference Targeting
1. Identifies devices emitting acoustic signals based on WiFi behavior
2. Targets specific devices for high-precision interference
3. Applies focused WiFi signals to manipulate device hardware
4. Permanently disables target devices through internal stress
